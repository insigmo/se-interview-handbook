# Полезные ресурсы для Инженеров

Этот документ поможет тем, кто выбрал путь, ведущий к хорошо оплачиваемой работе. Мы сгруппировали большой объём знаний и материалов, собранных из интернета **и с множества реальных собеседований**. Единственная просьба — поддержка: присылайте вопросы или записи ваших собеседований. Всё содержимое предоставляется бесплатно и создано благодаря тем, кто делился опытом раньше.

В первую очередь вам стоит начать с [ликбеза](docs/FromZeroToHero%20in%20QA%20%28AQA%29%20Interviews.pdf) по подготовке к собеседованиям, с нуля, начиная с оформления резюме, общения с HR, обсуждению большего ЗП и до первых офферов. 

Далее идут типовые вопросы собеседований. Вы можете создать свой документ с вопросами или отправить их Беталу Бербекову (Python) или Астемиру Балкарову (тестирование). Если не хотите выписывать вопросы — присылайте видео собеседований. Это минимальная отдача за полученный материал, ведь он создан благодаря тем, кто делился вопросами раньше.
